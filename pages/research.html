    Advancing Code Review and Code Refinement Automation Using Large Language Models

    Undergraduate Thesis, NLP Group, CSE-BUET
    Supervisors : Dr. Anindya Iqbal(Professor, BUET-CSE) Dr. Toufique Ahmed (PostDoc, UC Davis) Research Grant @ RISE - BUET
    Abstract Manuscript under preparation

  • We studied the efficacy of language models, in automating code review and code refinement activities using few-shot prompting and Parameter-efficient Fine-tuning.

    We augmented static program metadata (function call graph) and natural language summary (produced with codeT5-base) with human evaluation to improve code review and code refinement generation tasks. Few-shot examples are retrieved from the training dataset using BM-25 ranking algorithm.

    We used GPT-4o, GPT-3.5-turbo, Gemini-1.0-pro for prompt engineering and Llama-3, Code-Llama for fine-tuning. We collected the data from the Codereviewer dataset of Microsoft Research.

    We were able to surpass state-of-the-art pretrained model CodeReviewer by achieving over 90% BLEU in this task.

  • NLP X Systems Security

    Feb 2024 - Present

    Detecting Attack Behavior in Computer utilizing LLMs

    Supervisors : Dr. Md. Shohrab Hossain (Professor, CSE, BUET) Dr. Shahrear Iqbal (Research Officer, National Research Council, Canada)

  • Currently, We are investigating the prediction accuracy of Large Language Models (LLMs) in detecting attack behaviors within computer system logs.

    Our approach involves constructing provenance graphs and reverse trees (children-to-parent relationships). Subsequently, we generated event traces and then classified them. We labeled the data as either benign or malicious, aiming to supervise LLMs for precise detection of attack patterns.

    The logs used in this study are sourced from the DARPA OPTC and DARPA TC E3 datasets and are structured within a PostgreSQL database.

    Low Resource 2D Image Style Tran

    Supervisor : Sheikh Azizul Hakim (Lecturer, CSE, BUET)

  • We generate a new image by combining the content of one image with the style of another image while optimizing for minimal computational resource usage. We utilized Neural style transfer models : VGG16 and ResNet-50 and GAN model : CycleGAN.
    The output from the decoder is passed to VGG from which we extract features and calculate style loss and content loss. Then we calculate perceptual loss from the weighted sum of Style loss and content loss. For content loss, higher layers are used. For style loss, lower layers of networks are used. In defining our loss functions according to our content and style image shapes, we utilized the Gram Matrix by which we could obtain the MSE loss between content and style images. We utilized FID, SSIM, and PSNR as evaluation metrics.
    With the help of these models, we are able to turn any picture into a painting of any famous artist. Paintings by Monet (Giverny in springtime), Pissarro (Chestnut trees, Louveciennes, Spring - 1870), and Van Gogh(Starry Night) are used for result picture. Now, we are trying to transfer the style of calligraphy to a text image.
