    Advancing Code Review and Code Refinement Automation Using Large Language Models

    Veritatis et quasi architecto beatae vitae dicta sunt, explicabo.

  • My undergraduate thesis project was under Dr. Anindya Iqbal and Dr. Toufique Ahmed (PostDoc, UC Davis). We studied the efficacy of language models, in automating code review and code refinement activities using few-shot prompting and Parameter-efficient Fine-tuning. We augmented static program metadata (function call graph) and natural language summary (produced with codeT5-base) with human evaluation to improve code review and code refinement generation tasks. Few-shot examples are retrieved from the training dataset using BM-25 retrieval algorithm. We used GPT-4o, GPT-3.5-turbo, Gemini-1.0-pro for prompt engineering and Llama-3, Code-Llama for fine-tuning. We collected the data from the Codereviewer dataset of Microsoft Research. We were able to surpass state-of-the-art pretrained model CodeReviewer by achieving over 90% BLEU in this task. At present, we are preparing the manuscript of our work for publication.
