  • Ground-Water-Level-Prediction

    View on Github

    Python,seaborn, scipy numPy, Pandas

    At first, we preprocess the data, then used many types of regression models to predict the ground level water in different regions of Bangladesh to install deep tube wells. For seasonal forecasting, we are using ARIMA, LSTM and GRU models. It is a funded research project by WARPO (Water Resources Planning Organization) under the Ministry of Water Resources, Government of Bangladesh. Supervisor: Dr. A. B. M. Alim Al Islam (Professor, CSE, BUET)

  • Vehicle Object Detection in context of BD traffic

    View on Github

    Yolo, Faster-RCNN, rtDETR, Transfer-Learning

    We address complex task of object detection in driving environments across 9 districts in Bangladesh using BadODD dataset. We employed transfer learning using three different approaches, such as : YOLOv6 and YOLOv8, Novel transformer-based models such as rtDETR and CoDETR. We also explored Faster R-CNN. It was a part of SUST Deep Learning Enigma competition in the field of Computer Vision, where we secured 8th position among around 100 teams.

  • Machine Learning Algorithms and Neural Network from Scratch

    View on Github

    Python, numPy, Pandas

    I developed and applied advanced machine learning techniques, including matrix transformations, eigenvalue decomposition, and image reconstruction via singular value decomposition. I implemented AdaBoost with logistic regression for multi-class classification, built a feedforward neural network from scratch with various activation functions and regularization techniques, and performed PCA and clustering using Gaussian Mixture Models and the EM algorithm.
